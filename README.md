# AI Powered Quiz App

A modern quiz application developed using React.js featuring:

- Interactive quiz system
- Countdown timer
- Score tracking
- Multiple questions
- Restart functionality
- Responsive UI design

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
- Git & GitHub

## Features
- Real-time quiz interaction
- Dynamic question management
- Final score summary
- User-friendly interface
